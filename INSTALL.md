## 1	GOD Intallation1. linux or unix install2. Mac ### 1.1	Preper#### 1.1.1	for linux or unixDependency Build Instructions(ubuntu)		sudo apt-get update 	sudo apt-get install build-essential libtool autotools-dev automake pkg-config libssl-dev libevent-dev bsdmainutils	sudo apt-get install libboost-all-dev	sudo apt-get install software-properties-common	sudo add-apt-repository ppa:bitcoin/bitcoin	sudo apt-get update	sudo apt-get install libdb4.8-dev libdb4.8++-dev	sudo apt-get install libminiupnpc-dev#### 1.1.2	for macinstall OS X command line lib:xcode-select --install install [Homebrew](https://brew.sh/).install homebrew related dependency:			brew install automake berkeley-db4 libtool boost --c++11 miniupnpc openssl pkg-config protobuf libevent### 1.2 build Bitcoin God Core1.download the GOD source code and go to bitcoin folder		git clone https://github.com/tokengod/godcoind.git 	cd godcoind 2.build Bitcoin God Core: conﬁgure，--without-gui 		./autogen.sh 	./configure --without-gui 	make 	make install### 1.3	Start GODbitcoin God can use for now. pls create rpc config file before you run it at first time, as following for `MAC`	echo -e "rpcuser={username}\nrpcpassword={password}" > "/Users/${USER}/Library/Application Sup port/Bitcoin/bitcoin.conf"	chmod 600 "/Users/${USER}/Library/Application Support/Bitcoin/bitcoin.conf"

	as following for `Linux`	echo -e "rpcuser={username}\nrpcpassword={password}" > "/${USER}/.bitcoin/bitcoin.conf"	chmod 600 "/${USER}/.bitcoin/bitcoin.conf"

	Start GOD		bitcoingodd -daemonif you wanto check the log, pls check debug.log file		tail -f {datadir}/debug.log
	
**Note**：if you have not point datadir when you start god, the log file will save to bitcoin.conf file same folder.	------------------------------------------------------------------------------------ ## 2	GOD Command### 2.1	Start and help- bitcoingodd -datadir={datadir full path} -conf={config file
